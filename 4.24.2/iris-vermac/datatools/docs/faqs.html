1. What is Java Web Start?

Java Web Start is a program created and distributed by Sun Microsystems for deploying applications via the Internet.

2. Why do I need Java Web Start?

Java Web Start serves a variety of functions. Those which are important in the context of DataTools are:
  • Version control. You will always be running the most current version of the software.
  • Increased speed. By caching the application to the local disk, application speed is improved versus loading the application from a remote location.

3. Why do I get an error dialog when trying to run DataTools?

There are a few common reasons that DataTools may fail to launch. Please check the following list for resolutions to common problems.
  • Download Error
  • Launch Error

4. How can I run the DataTools without a network connection?

  • Download the datatools zip file.
  • Unzip the archive to the local disk.
  • Create a directory named "traffic" at the root of the local disk.
  • Create subdirectories for each year that you wish to analyze data.
  • Copy the required .traffic files to the appropriate directory. The files can be downloaded from the University of Minnesota - Duluth
  • Double-click on the batch file in the distribution folder to launch the application.

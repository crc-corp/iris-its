Data Extract Download Error 

To start an application, JWS may need to download files from one of Sun Microsystems servers. If you get a Download Error, check the details and determine which of the following error types you are experiencing.

Failed downloading from Sun:

  1. Open the Java Web Start Application Manager ( start menu --> programs --> Java Web Start )
  2. Open the Preferences dialog ( found on the file pull-down menu )
  3. Select the "General" tab.
  4. Make sure that the proxy settings are correct for your location.
    5. You may have to contact your MIS support staff if you have questions on how to do this.

Data Extract Launch Error 

Launch errors can often be attributed to not having the correct Java Runtime Environment. Java Web Start is designed to install the correct version on demand, however, there have been issues with this process. To get around it, do the following:
  1. Install JRE 1.4.1 in "C:\jre1.4.1" by choosing "custom installation". You will need local administrator priviledges to do this.
  2. Open the Java Web Start Application Manager ( start menu --> programs --> Java Web Start )
  3. Open the Preferences dialog ( found on the file pull-down menu )
  4. Select the "Java" tab.
  5. Click the "Find" button.
  6. Click "Next".
  7. Browse to c:\j2re1.4.1 and click "Next".
  8. The JRE Finder will show you that it found a JRE, click "Next".
  9. Click "OK" on the "Preferences" dialog.
  10. Try to launch DataExtract.
  11. If this does not fix the problem, call the DataExtract support person list on the DataExtract Home Page
